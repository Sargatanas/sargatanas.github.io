ФОРМУЛЫ НА MATHML

Получить формулу средствами MathML согласно варианту.

2 - x - cos  x + sin 2 x y

Справка

В стандарте HTML5 язык MathML распознается внутри обычной страницы не во всех браузерах. Чтобы он заработал, требуется в тег head включить следующий скрипт:

<​script>
window.MathJax = { MathML: { extensions: ["mml3.js", "content-mathml.js"]}};
<​/script>
<​script type="text/javascript" async src="//cdn.mathjax.org/mathjax/latest/MathJax.js?config=MML_HTMLorMML">
<​/script>

P.S. Впрочем, старым версиям IE не помогает даже это...
